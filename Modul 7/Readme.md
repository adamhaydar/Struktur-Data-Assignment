## Dasar Teori

Stack merupakan struktur data yang menerapkan prinsip LIFO (Last In, First Out), di mana elemen yang paling akhir dimasukkan akan dikeluarkan terlebih dahulu.

## Guided 

### 1. [Nama Topik]

```C++
#include <iostream>
using namespace std;

int main() {
    cout << "ini adalah file code guided praktikan" << endl;
    return 0;
}
```
Stack diimplementasikan dengan memanfaatkan array dan menyediakan operasi dasar seperti push dan pop, serta dilengkapi fungsi tambahan untuk memperbarui dan mencari data.
